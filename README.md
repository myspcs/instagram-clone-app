# Instagram Clone (MERN)

Bu proje, MERN yapısıyla hazırlanmış basit bir Instagram klonudur. Kullanıcılar hesap oluşturabilir, giriş yapabilir, fotoğraf paylaşabilir, gönderileri beğenebilir ve mesajlaşabilir. Arayüz React ile, sunucu Express ile, veriler ise MongoDB üzerinde tutulur.

## Kurulum
- backend ve frontend klasörlerinde `npm install` çalıştırın.
- frontend için `npm start`, backend için `npm run dev` komutlarını kullanın.

## Özellikler
- Kullanıcı kayıt ve giriş
- Fotoğraf yükleme ve gönderi paylaşma
- Beğeni sistemi
- Profil sayfası
- Gerçek zamanlı sohbet
